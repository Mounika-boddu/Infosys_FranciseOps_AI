# Milestone 1 – Login, Signup & Forgot Password System

## Project Overview
This project implements a secure authentication system using Streamlit.

## Features
- Login
- Signup
- Forgot Password
- JWT Authentication
- User Dashboard
- Admin Dashboard

## Tech Stack
- Python
- Streamlit
- Google Colab
- JWT
- Gmail SMTP
- ngrok

## How to Run
1. Open the notebook in Google Colab.
2. Install the required libraries.
3. Add Colab Secrets.
4. Run all notebook cells.
