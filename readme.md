# Milestone 1 – Login, Signup & Forgot Password System

## Project Overview
This project is the first milestone of the Franchise Analytics & Management internship. It implements a secure user authentication system using Streamlit. The application includes Login, Signup, Forgot Password, JWT-based session management, OTP email verification, and an Admin Dashboard.

## Features
- User Login
- User Signup
- Forgot Password using Security Question
- Forgot Password using Email OTP
- JWT-based Session Handling
- User Dashboard
- Admin Dashboard
- Email and Password Validation
- Logout Functionality

## Tech Stack
- Python
- Streamlit
- Google Colab
- JWT (JSON Web Token)
- Gmail SMTP
- ngrok

## How to Run
1. Open the notebook in Google Colab.
2. Install all required Python libraries.
3. Add the required secrets in Colab Secrets:
   - JWT_SECRET
   - NGROK_AUTHTOKEN
   - EMAIL_ADDRESS
   - EMAIL_PASSWORD
4. Run all notebook cells.
5. Open the ngrok public URL to access the Streamlit application.
